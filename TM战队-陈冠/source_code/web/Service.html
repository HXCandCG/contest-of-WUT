用户须知

如何试用该系统

本店所有服务只针对武昌理工学子，无关人等请退出系统。

如何点餐？

点击相应菜品板块，+视为选中，点击-则取消选项。

如何填写地址？

在注册时地址已经自动生成所有地址信息以注册时为准。地址可以在点餐页面进行修改！

如何支付？

在选好菜品和地址后点击最下方支付下单按钮，订单自动生成。

几点声明

1.本店所有煲仔饭全部由三食堂原煲仔饭窗口提供，食堂菜品保证健康与质量，所有菜品均平价出售，页面收入来源只针对配送费（不包括饮品和烟）。

2.本店配送费以商品费用递增十元以下一律0.5￥,超出部分每十元增加0.5￥，配送费最高5￥超出部分配送费不变！

3.本店提供的配送服务是直接前往具体寝室配送，若地址填写时楼号或门牌号填错，本店概不负责，若配送出错，本店将回应适当的补偿。

4.由于对餐量需求比较集中，所以配送员配送期间，若客户本人在寝室内，则配送员将把物品交到客户手中才可离开， 若客户不再寝室，为了不影响其他客户等餐时间，配送员只需把菜品挂在门把上，并拍下照片证明即可，请各位客户谅解。

5.本服务页面暂时仅支持微信支付，若带来不便请谅解，当支付完成后请稍做等待，若支付结束后返回了主页则支付成功，若停留在了当前支付页面请及时与我们联系。

6.一旦下单支付不予退款，若对当次服务有疑问或者意见的可以在本微信公众号界面直接留言或者直接拨打下方电话号码，我们一定及时予以反馈。
